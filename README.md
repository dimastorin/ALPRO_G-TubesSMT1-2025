## Template Penamaan File

**Format Nama File:**
```
NIM_NamaLengkap_JudulTubes.py
```

**Contoh:**
```
11251015_DesyAdelinaR_FlashcardStudyManager.py
11251020_BudiSantoso_SistemKasir.py
11251025_SitiNurhaliza_AplikasiTodoList.py
```

**Ketentuan Penamaan:**
- Gunakan **NIM** tanpa spasi
- Gunakan **Nama Lengkap** tanpa spasi (PascalCase/CamelCase)
- Gunakan **Judul Tubes** tanpa spasi (PascalCase/CamelCase)
- Pisahkan dengan underscore (`_`)
- Ekstensi file `.py`
